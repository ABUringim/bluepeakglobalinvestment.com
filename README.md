# Blue Peak Global Investment Limited Website

A professional, multi-page website for Blue Peak Global Investment Limited showcasing their multi-sector investment services.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations
- **7 Core Services**: Dedicated pages for each service sector
- **AI Chat Assistant**: Interactive chatbot on every page
- **Auto-Scrolling Gallery**: 21-image seamless gallery on homepage
- **Contact Form**: Integrated contact form with Google Maps
- **Sticky Navigation**: Easy navigation across all pages

## Pages

1. **Home** (index.html) - Hero section, scrolling gallery, services overview
2. **About Us** (about.html) - Company vision, mission, CEO message
3. **Services** (services.html) - Detailed overview of all services
4. **Contact** (contact.html) - Contact form and Google Maps integration
5. **Service Detail Pages** (7 pages) - In-depth information for each service:
   - Trading & Exploration of Mineral Resources
   - Oil, Gas & Energy Marketing
   - Import & Export
   - General Production
   - ICT & Electronics
   - General Contract & Trading
   - Multi-Sector Operations

## Setup Instructions

1. **Extract Files**: Ensure all files are in the correct directory structure
2. **Add Images**: Place your images in the `images/gallery/` folder (21 images required)
3. **Open in Browser**: Open `index.html` in any modern web browser
4. **Customize**: Edit content in HTML files as needed
