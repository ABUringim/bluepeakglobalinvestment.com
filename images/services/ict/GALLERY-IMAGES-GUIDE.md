## Image Specifications

### Technical Requirements:
- **Format**: JPG (JPEG)
- **Dimensions**: 800x600 pixels minimum (4:3 ratio recommended)
- **File Size**: 200-500KB each (optimized for web)
- **Quality**: High resolution, professional photography
- **Orientation**: Landscape (horizontal)

### Style Guidelines:
- Professional, high-quality images
- Good lighting and composition
- Relevant to the specific service
- Diverse representation of people
- Modern, clean aesthetic
- Consistent quality across all images

## Quick Download Process

1. Visit Unsplash.com or Pexels.com
2. Search using the keywords provided above
3. Select high-quality, relevant images
4. Download in original size
5. Rename according to the filename specified
6. Optimize/compress if over 500KB
7. Place in the correct folder

---

## Image Optimization Tools

### Online (Free):
- **TinyJPG** - https://tinyjpg.com
- **Compressor.io** - https://compressor.io
- **Squoosh** - https://squoosh.app

### How to Optimize:
1. Upload your downloaded image
2. Compress to 70-80% quality
3. Ensure file size is under 500KB
4. Download optimized version
5. Verify image still looks professional

## Gallery Features

### Interactive Lightbox:
- Click any image to open in full-screen lightbox
- Navigate with arrow buttons or keyboard
- Close with X button or ESC key
- Displays image caption below

### Responsive Design:
- **Desktop**: 4 images per row
- **Tablet**: 2 images per row
- **Mobile**: 1 image per row (stacked)

### Hover Effects:
- Slight zoom-in effect
- Shadow enhancement
- Smooth transitions
