## Step-by-Step Download Instructions

### For Unsplash:
1. Click the search link above
2. Browse through images
3. Click on image you like
4. Click "Download free" button (top right)
5. Image downloads automatically
6. Rename to match required filename (e.g., mining-1.jpg)
7. Move to `images/gallery/` folder

### For Pexels:
1. Visit pexels.com
2. Search using terms above
3. Click on image
4. Click "Free Download" button
5. Select "Original" size
6. Rename and move to gallery folder

### For Pixabay:
1. Visit pixabay.com
2. Search using terms above
3. Click on image
4. Click "Free Download"
5. Select size (1920x1280 recommended)
6. Rename and move to gallery folder

## Image Specifications

### Required Format
- **File Type**: JPG (JPEG)
- **Dimensions**: 1200x900 pixels minimum (4:3 ratio)
- **File Size**: Under 500KB each (compress if needed)
- **Quality**: High resolution, professional

### Image Optimization Tools

**Online (Free):**
- TinyJPG - https://tinyjpg.com
- Compressor.io - https://compressor.io
- Squoosh - https://squoosh.app

**How to optimize:**
1. Upload your image
2. Download compressed version
3. Should be under 500KB
4. Quality should still look professional
