## What You Need to Do

### Step 1: Create Folder Structure
```
images/
└── services/
    ├── minerals/
    ├── energy/
    ├── import-export/
    ├── production/
    ├── ict/
    ├── contract/
    └── multi-sector/
```

### Step 2: Download Images
- **Total needed**: 56 images (8 per service)
- **Sources**: Unsplash, Pexels, Pixabay
- **Format**: JPG
- **Size**: 1200x900px recommended
- **File size**: Under 300KB each

### Step 3: Name & Place Images
Use exact filenames from GALLERY-IMAGES-COMPLETE-GUIDE.md

**Example for Minerals:**
- core-sample.jpg
- mining-operations.jpg
- resource-mapping.jpg
- mineral-portfolio.jpg
- site-operations.jpg
- geological-surveying.jpg
- quality-assessment.jpg
- logistics-export.jpg

### Step 4: Test
1. Open any service page (e.g., service-minerals.html)
2. Scroll to "Our Work in Action" section
3. Click any image to open lightbox
4. Test navigation arrows
5. Test on mobile device
