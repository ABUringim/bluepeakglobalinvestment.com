# Quick Start Guide

## Getting Your Website Running in 3 Steps

### Step 1: Download Images
1. Visit free stock photo websites:
   - [Unsplash](https://unsplash.com)
   - [Pexels](https://pexels.com)
   - [Pixabay](https://pixabay.com)

2. Search for and download 21 images (3 per category):
   - **Mining**: "mining operations", "mineral extraction"
   - **Oil & Gas**: "oil rig", "gas processing", "energy plant"
   - **Logistics**: "cargo ship", "container port", "warehouse"
   - **Production**: "manufacturing", "factory", "production line"
   - **ICT**: "data center", "server room", "technology"
   - **Contract**: "construction site", "building project"
   - **Multi-sector**: "global business", "world trade", "logistics"

3. Rename and place them in `images/gallery/` folder as specified in the README.txt

### Step 2: Open the Website
1. Navigate to the website folder
2. Double-click `index.html`
3. The website will open in your default browser

### Step 3: Test All Features
- Click through all navigation links
- Test the AI chat widget (bottom right)
- Try the contact form
- View all 7 service detail pages
- Check mobile responsiveness (resize browser window)

## Optional Customizations

### Update Company Logo
- Replace the text logo in the header with an image logo
- Edit the `.logo` section in `css/styles.css`

### Change Colors
- Edit color variables in `css/styles.css` (lines 8-14)
- Update `:root` CSS variables

### Modify Contact Information
- Update footer in all HTML files
- Change contact details in `contact.html`

### Add Google Maps
- Get your Google Maps embed code
- Replace the iframe src in `contact.html` (line 145)

## Hosting Your Website

### Option 1: Free Hosting (GitHub Pages)
1. Create a GitHub account
2. Create a new repository
3. Upload all files
4. Enable GitHub Pages in repository settings

### Option 2: Web Hosting Service
1. Choose a hosting provider (Hostinger, Bluehost, etc.)
2. Purchase domain name
3. Upload files via FTP or cPanel File Manager
4. Point domain to hosting

### Option 3: Local Testing
- Simply open `index.html` in any browser
- No server required for basic functionality

## Troubleshooting

**Images not showing?**
- Check that images are in the correct folder
- Verify image file names match exactly
- Ensure images are in JPG format

**Chat widget not working?**
- Check browser console for errors (F12)
- Ensure `js/script.js` is loaded correctly

**Styling looks broken?**
- Verify `css/styles.css` is in the correct location
- Check that Font Awesome CDN is accessible
- Clear browser cache (Ctrl+F5)

**Mobile menu not working?**
- Ensure JavaScript is enabled in browser
- Check that `js/script.js` is loaded
