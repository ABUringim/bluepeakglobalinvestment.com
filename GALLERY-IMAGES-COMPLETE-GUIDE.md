# Complete Gallery Images Guide

## Overview
Each of the 7 service pages now includes an "Our Work in Action" gallery section with 8 professional images and a lightbox feature for viewing.

## Total Images Required: 56 Images
- 8 images per service page × 7 service pages = 56 images

## Image Specifications
- **Format**: JPG
- **Dimensions**: 1200x900 pixels (4:3 ratio) recommended
- **File Size**: Under 300KB each (optimized)
- **Quality**: High resolution, professional photography

## Folder Structure

Create the following folder structure in your `images/` directory:

```
images/
├── services/
│   ├── minerals/
│   ├── energy/
│   ├── import-export/

│   ├── ict/

```

## Gallery Features Implemented

### Interactive Lightbox:
✅ Click any image to view full-size
✅ Navigate with Previous/Next arrows
✅ Navigate with keyboard (Left/Right arrows)
✅ Close with X button or Escape key
✅ Click outside image to close
✅ Smooth transitions and animations

### Responsive Design:
✅ Desktop: 4 images per row
✅ Tablet: 2-3 images per row
✅ Mobile: 1 image per row (stacked)
✅ Hover effects (zoom and shadow)
✅ Touch-friendly on mobile devices

### Professional Styling:
✅ Captions below each image
✅ Consistent spacing and alignment
✅ Matches website color scheme
✅ Smooth animations
✅ Professional appearance
